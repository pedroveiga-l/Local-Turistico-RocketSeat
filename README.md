# Local Turístico — Busan

Projeto desenvolvido como **desafio prático de HTML e CSS** da **Formação Full Stack da Rocketseat**.

O objetivo do desafio é recriar fielmente, a partir de um **layout fornecido no Figma**, uma página estática apresentando um local turístico, aplicando boas práticas de **estrutura semântica**, **estilização com CSS** e **organização de código**.


## Sobre o projeto

A página apresenta a cidade de **Busan**, na Coreia do Sul, destacando seus principais pontos turísticos, história e paisagens. Todo o conteúdo foi estruturado com foco em:

* Fidelidade visual ao layout do Figma
* Código limpo e organizado
* Uso correto de HTML semântico
* Estilização moderna e responsiva

Este projeto tem caráter **educacional**, com foco em consolidar fundamentos essenciais do Front-end.


## Tecnologias utilizadas

* **HTML5**
* **CSS3**

## Aprendizados

Com este desafio, foi possível praticar:

* Estruturação de páginas com **HTML semântico**
* Organização e reutilização de estilos no **CSS**
* Posicionamento de elementos
* Atenção a detalhes visuais e espaçamentos
* Leitura e interpretação de layouts no Figma
